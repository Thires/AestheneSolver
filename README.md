### Clone the repository:

# On Linux:

# On macOS:
Tkinter is usually preinstalled. If not, install Python from:

https://www.python.org/downloads/mac-osx/

Or, if using Homebrew Python:
```bash
brew install python-tk
```

# Run it
python AestheneSolver.py
